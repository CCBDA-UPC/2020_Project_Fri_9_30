### Explaination of dataset
ID: Spanish phone number by random generation(Unique)

Name: Last name by random generation(not unique, name may be same)

Gender: 1 Male 0 Female by random generation

Occupation: 1-50 means tags of important occupation list by random generation. For example, 23 = Software Engineer

Income: Income is based on the Spain average salary by standard normal distribution and actual meaning adjustment

District:The main districts in Barcelona 
1=Ciutat Vella
2=L'Eixample
3=Sants-Montjuïc
4=Gràcia
5=Les Corts
6=Sarri Sant Gervasi
7=Horta Guinard
8=Nou Barris
9=Sant Andreu
10=Sant Mart
Latitude: Barcelona 41.3851° N, 2.1734° E

Longtitude：41.3851° N, 2.1734° E

Timestamp：From 001-100 For example, 001=28/04/2020

Health Care Capacity: Control approaches.
1=Business As Usual 
2=Reduced Interaction 
3=Lock-Down 
4=Self-Isolation

Health Status: 
1=Healthy 
2=Infected 
3=Recovered 
4=Dead
